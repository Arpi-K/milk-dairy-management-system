# 🥛 Milk Dairy Management System

A web-based Milk Dairy Management System developed using PHP and MySQL to simplify dairy business operations. The system helps manage customer records, milk collection details, payments, and overall dairy management efficiently.

## 📖 Overview

The Milk Dairy Management System is designed to automate the daily activities of a dairy business. It reduces manual record-keeping and provides an organized way to manage customer information, milk collection, and payment records.

## ✨ Features

- Customer Registration and Management
- Milk Collection Tracking
- Payment Management
- Dairy Record Management
- Admin Dashboard
- Database Integration
- User-Friendly Interface

## 🚀 Installation

### Prerequisites

- XAMPP/WAMP
- PHP
- MySQL

### Steps to Run

1. Clone the repository

```bash
git clone https://github.com/Arpi-K/milk-dairy-management-system.git
```

2. Move the project folder to:

```text
xampp/htdocs/
```

3. Start Apache and MySQL from XAMPP Control Panel.

4. Open phpMyAdmin.

5. Create a database.

6. Import the provided SQL file.

7. Configure database connection settings if required.

8. Open your browser and run:

```text
http://localhost/milk-dairy-management-system
```
